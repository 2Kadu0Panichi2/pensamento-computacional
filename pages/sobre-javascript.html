O que é JavaScript e Para Que Serve?


A origem do JavaSript


O JavaScript é uma linguagem de programação de alto-nível, criada em 1996 pelo programador Brendan Eich que, além de criar o JavaScript, foi também um dos fundadores da Mozilla Corporation.


Ele criou a linguagem quando ainda trabalhava na empresa Netscape, sendo projetada para rodar no Netscape Navigator, seu objetivo era oferecer aos desenvolvedores formas de tornar determinados processos de páginas web mais dinâmica, tornando seu uso mais agradável.


Embora essa linguagem tenha esse nome, vamos deixar claro que Javascript não é Java. Apesar da confusão que as pessoas normalmente fazem sobre essas duas linguagens, Java e Javascript não possuem nenhuma ligação direta uma com a outra a não ser o java no nome. Antes, a linguagem recebeu nomes como LiveScript e Mocha, mas, para aproveitar o grande sucesso da linguagem Java no mercado, os executivos da Netscape resolveram mudar o nome para JavaScript.


O que é JavaScript e como funciona


O JavaScript, ou também conhecido como, JS, é uma linguagem de programação de alto-nível usada para desenvolver aplicações, sistemas e serviços de alta complexidade. Com ela, você pode criar páginas dinâmicas, animações, gráficos, mapas interativos, aplicativos para dispositivos móveis e até games.


Foi criado como uma linguagem de programação client-side. Ou seja, é uma linguagem que é executada no computador do próprio usuário. Quem vai processar essa linguagem não é o servidor, mas sim o seu navegador.


Como todo o código JavaScript está no próprio navegador, o usuário pode ver o código e manipulá-lo. Isso faz com que as linguagens client-side sejam inseguras para fazer coisas como acessar um banco de dados, por exemplo. Por isso temos as linguagens server-side que atuam no lado do servidor, o backed. Como o PHP, Java, .NET, Python etc.


Sendo assim, o JavaScript atua como um complemento às linguagens HTML e CSS no front-end. Criei outros artigos falando sobre o que é HTML e o que é CSS. Eles vão ajudar você a ter um entendimento melhor sobre o que é JS e como funciona o desenvolvimento de um site. Mas, mesmo assim, vou explicar resumidamente cada uma dessas “camadas”.


  1. O que é HTML? – É uma linguagem de marcação de texto que serve para dar significado e estrutura de uma página. Permitindo definir parágrafos, cabeçalhos, inserção de imagens e vídeos.
  2. O que é CSS? – É uma linguagem de folha de estilos, especificando o estilo do layout de uma página. Ou seja, é tudo relacionado à aparência de um site.
  3. O que é JavaScript? – É uma linguagem de programação que controla o comportamento dos elementos de uma página. Executando funções dinâmicas dos conteúdos, animações e controle de mídias para deixar seu site mais interativo e interessante.

As vantagens do JavaScript


Saber como programar em JavaScript irá trazer uma série de vantagens ao desenvolvedor. Abaixo estão apenas alguns dos benefícios de usar essa linguagem:


  1. É mais fácil de aprender do que as outras linguagens de programação;
  2. Você não precisa de um compilador porque os navegadores interpretam a linguagem por conta própria.
  3. É mais leve e rápida que outras linguagens de programação.
  4. É totalmente compatível com vários navegadores e plataformas;
  5. Você pode validar entradas e reduzir a necessidade de verificações manuais de dados;
  6. Executa comportamentos específicos em uma página, como cliques e efeitos personalizados.

As desvantagens do JavaScript


Todas as linguagens de programação têm suas desvantagens, e o JavaScript não é a exceção. Veja agora alguns desses pontos fracos do JS:


  1. É vulnerável a brechas de segurança;
  2. Pode ser usada para executar programas maliciosos sem que o usuário saiba;
  3. Pode ser renderizada de maneira diferente pelos dispositivos compatíveis, causando problemas de desempenho.
